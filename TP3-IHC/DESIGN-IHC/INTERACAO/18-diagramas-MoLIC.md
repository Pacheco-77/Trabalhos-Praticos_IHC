# 18. Modelagem de Interação — Diagramas MoLIC 
Os Diagramas MoLIC são usados para representar o diálogo entre o usuário e o sistema, mostrando como a interação acontece passo a passo. Essa abordagem entende o uso da interface como uma “conversa”, permitindo visualizar intenções, ações e respostas. Com isso, o MoLIC ajuda a identificar problemas de interação e a melhorar a forma como o sistema guia o usuário durante as tarefas. É uma ferramenta útil para projetar interfaces mais claras e eficientes.

## Lembrete 

#### Diagrama MoLIC
O diagrama mostra o fluxo de interação do usuário dentro do aplicativo até visualizar seus lembretes. Primeiro, o usuário entra no app e realiza o login. Se os dados estiverem corretos, o sistema permite avançar; caso contrário, retorna uma mensagem de erro e volta para a tela de login. Após o login válido, o usuário acessa a tela de seleção de lembretes, onde pode escolher qual lembrete deseja visualizar ou consultar medicamentos pendentes. Ao prosseguir, o sistema exibe os lembretes correspondentes, mostrando as informações necessárias para que o usuário acompanhe suas medicações. O diagrama também mostra a possibilidade de voltar para a tela anterior quando necessário.
