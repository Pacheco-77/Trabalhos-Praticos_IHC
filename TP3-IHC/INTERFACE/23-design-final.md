# 23. Design Final

## Tela de Login
Na tela de login o usuário terá a possibilidade tanto para logar no app diretamente quanto para selecionar a interface com acessibilidade e cadastrar-se na plataforma.

<img width="440" height="956" alt="login" src="https://github.com/user-attachments/assets/cf8e533b-63c4-4c51-8c07-d036d7d157a7" />

## Tela Inicial
Enquanto na tela inicial do app, o usuário terá acesso a função de adicionar lembretes, rotina, medicamento e um contato direto com o seu cuidador.

<img width="440" height="956" alt="INICIAL APP" src="https://github.com/user-attachments/assets/d20837c1-4112-4057-8a9d-1091be219ac0" />

## Tela de Lembretes
Na tela de lembretes, o usuário terá acesso às suas principais atividades do dia (especialmente para usuários com problema de Alzheimer), para ajudar o mesmo a realizar coisas do seu cotidiano.

<img width="440" height="1206" alt="LEMBRETES" src="https://github.com/user-attachments/assets/d7d4b2b6-1c04-40e6-a9e6-44e2f373cb1f" />

## Tela do Cuidador
Uma das telas mais importante do aplicativo, a qual o terá um botão para realizar ligação direto para o cuidador do Idoso.

<img width="440" height="1365" alt="CUIDADOR" src="https://github.com/user-attachments/assets/b930a077-98aa-474c-9a4b-ba2a469607ff" />

## Tela de Cadastro de Medicamento
Nessa tela será possível cadastrar o medicamento, sua dose e o horário de tomar o remédio.

<img width="440" height="956" alt="CADASTRO MEDICAMENTO" src="https://github.com/user-attachments/assets/4284ceff-14f6-4dd2-b359-e1c39e94161d" />

Na tela e lista de medicamentos, será exibido os medicamentos pendentes de tomar, medicamentos já tomados e medicamentos que ainda irão ser tomados.
